The Arduino based projects include,

Automatic lights using IR and PIR sensors, 
Blind Stick using ULTRASONIC sensor,  
Flex sensor based led, 
LDR with led, 
LM35 with pushbutton and led, 
Object Avoidance using ULTRASONIC sensor, 
People count using ULTRASONIC, 
Automatic street lights using IR sensor, 
Secure Alarm using PIR and ULTRASONIC sensors, 
Smart Door using IR sensor, 
Automatic watering system using SOIL MOISTURE sensor.
